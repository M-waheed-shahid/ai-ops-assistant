# ai-ops-assistant

**If someone else downloads the repo, they should do this:**

Open the project folder in VS Code.

**Create a virtual environment:**

python -m venv .venv

**Activate it:**

.venv/Scripts/activate

**Install dependencies:**

if you have requirements.txt: python -m pip install -r requirements.txt

otherwise: python -m pip install streamlit altair pandas

In VS Code, select the .venv interpreter

**Run the app with:**

python -m streamlit run dashboard/app.py

That is the correct workflow for this Streamlit project.
